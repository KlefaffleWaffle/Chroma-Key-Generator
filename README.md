# Background-Generator
A program to set RGB values of the entire screen for the sake of Chroma Keying in video editing Software

# Problem Statement 1
Prosumer video editing tools like Final Cut, Adobe Premiere, and Davinci Resolve all use a tool known in the industry as "Chroma Key." While the various software options may have their own unique name for the tool and unique interfaces, they all solve the same problem. They find an RGB or range of RGB color values, and make those pixels transparent. This is often times used with green or blue screens. In photo editing, this is similar to the "Magic Wand" tool in Photoshop. The color green is often used in the industry because it has the highest contrast with skin tone. This leads to efficient editing as the computer can remove the background very easily while keeping the actors and everything else in the scene with minimal clean up and correction. However it creates a problem. You can't have anything of the same or similar color in the scene. If a character or object needs to be green, you need to use a separate color background. That's not the end of the world, except, most equipment designed for chroma key is either green or blue. As I was making a video and planning to do a lot of computer generated environments, I realized, neither green nor blue screens are an ideal choice because of my character's costume. 

# Solution
This software allows you to use your monitor as a background for your footage that can be any color value you like. You are not bound by the constraints of green or blue colors. That allows you to use both green and blue and many other colors simultaneously. While in practice, this only works for filming and photographing things that are smaller in scale, with larger consumer tvs, you can film and photograph something, with a max dimension of almost 4 feet. Even still we have seen similar technology being used in major motion pictures like Superman (2025).

# Problem Statement 2
Another issue of consumer green screens is that inconsistent lighting can cause severe issues while chroma keying. Generally to avoid this, you need proper lighting which also generally requires the screen to be flat to avoid shadows and discoloration. Proper lighting can be hard to achieve. For productions with minimal budget and crew, you might waste a whole day filming only to realize, the green screen is unusable and therefore the whole day is unusable. 

# Solution
This software solves that issue, because you can guarantee consistency in the colors. The screen emanates light rather than reflecting it, which means you have precise control over the exact RGB value being output. In practice, this proved more capable than anticipated. As I was filming a demonstration, the tool performed well enough to successfully film green on green.

# Problem Statement 3
Film equipment, especially mid to high quality equipment, is expensive. On Amazon, a large green screen can run $40 without a stand to hang it. A green screen with a stand can run $100-$200. Lights can easily run you another $80. That's $200-$300 just for the green screen. No actors, no camera, no editing software, no props, no costumes. That's just a green screen.

# Solution
This software only requires a monitor and a computer to run it on. The screen can be a computer monitor or a television, which many people already own.
